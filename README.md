# Once Framework
The build once, use everywhere framework

The Once framework lets you write fully portable and cross-platform applications for all modern web, mobile and desktop platforms using JavaScript and either the Angular or React ecosystem. This project adheres to the Contributor Covenant code of conduct. By participating, you are expected to uphold this code. Please report unacceptable behavior to admin@buildonce.co

## Installation

To install prebuilt Once binaries, use [`npm`](https://docs.npmjs.com/).
The preferred method is to install Once as a development dependency in your
app:

```sh
npm install once --save-dev
```
